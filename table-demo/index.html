Table Demo

A table is a way of creating a structured grid display of date. The paired tags(open/close) we will be interested in are:

  • <table> : identifies the start of the table
  • <thead> : identifies the header row of the table
  • <tbody> : identifies the data details body of the table
  • <tfoot> : identifies the foooter row of the table
  • <tr> : identifies the start of row on the table
  • <th> : identifies the cell on the head row for the column title
  • <td> : identifies the cell on the deatail row or footer to hold the content

Some tag have parameters. The border parameter of the table tag applies a border around your table. The colspan parameter and rowspan parameter are used on the <td> tab to allow you to merge column on a single row or columns cells across rows.
